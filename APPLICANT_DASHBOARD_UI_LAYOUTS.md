## Interactive Elements

### Buttons

-   **Primary**: Blue background, white text, hover darker
-   **Secondary**: Gray background, gray text, hover lighter
-   **Danger**: Red background, white text, hover darker
-   **Icon**: Emoji buttons with hover effect

### Tabs

-   **Active**: Bottom blue border, blue text
-   **Inactive**: No border, gray text, hover gray bg

### Filters

-   **Active**: Blue background, white text
-   **Inactive**: Gray border, gray text, hover gray bg

### Inputs

-   **Default**: Gray border, white bg
-   **Focus**: Blue ring, blue border
-   **Error**: Red border, red text
