## Features in Action

### Viewing All Threads

1. Navigate to `/community`
2. See list of all threads sorted by recent activity
3. See company logos, titles, previews, reply counts
4. Click any thread to open conversation

### Reading a Conversation

1. Click thread from list
2. See fixed header with company info
3. Read all messages in chronological order
4. See chat bubbles with proper styling
5. See timestamps for each message

### Posting a Message

1. Sign in (if not already)
2. Open a thread
3. Type message in bottom box
4. Click "Send"
5. Message appears instantly in conversation
6. Scroll auto-updates to show new message

### Creating a New Thread

1. Click "Start a Thread" button
2. Select company from dropdown
3. Enter thread title
4. Write your message
5. Click "Create Thread"
6. New thread appears in community list
7. Start receiving responses
